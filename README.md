# DD

### 适用于ididi的命令行工具。

#### 安装方法

`npm install ddcmd -g`

#### 具体命令

##### 1. dd pagelist

在使用ididi开发的h5端内项目中，各页面之间的联系不紧密，没有一个菜单去展示所有的页面，
dd pagelist命令就是将项目中的所有页面罗列出来，并且给出预览。这有利于我们对项目中页面的
整体了解。具体使用方法请执行`dd pagelist -h`

##### 1. dd pagelist

##### 2. dd pagetable

`dd pagelist`是只展示选择的页面，而`dd pagetable`是将项目中所有的页面都展示出来。具体使用方法请执行`dd pagetable -h`
